 instrucciones:
 Crear un programa sobre el juego del  gato en la imagenes indican la maneran
 en la que se gana y las cuales sinver para el analizes 
 de patrones para encontrar la manera que se puede ganar

 el programa debera de contar con lo siguientes:

 un menu estructura de la siguiente manera:
		1.-iniciar Juego.
		2.-Cargar partida.
		3.-Records
		4.-Salir.

los contenidos del menu al mostraro seran de la siguiente manera:

1.-Iniciar Juego.
	despues de dar inicio se creara un archivo con el nombre que indique el jugador
	luego habra un submenu  de la siguiente manera:
		1.-jugar con la maquina.
		2.-Jugar con otro usuario.
		3.-salir
	las descripciones de este submenu seran las siguientes:
		1.-al jugar con la maquina los movimientos pueden ser ya sea
		 hechos por probabilid o de manera aleatorian,
		 se deberan de tener las siguientes opciones
		 se debera de contar las partidas que se hayan jugado quienes 
		los haya ganado y  perdido 
		es decir que en una parte habra un indicador de ganadores y perdedores
		TAMBIEN en cada vez se se realize una accion se volvera a carga de numero 
		para que la pantalla no este alargando, igualmente habra una 
		opcion para guardar la partida que se este jugando.

		2.-al jugar con otro usuario los movimientos se preguntan cada vez,
		 se deberan de tener las siguientes opciones
		 se debera de contar las partidas que se hayan jugado quienes 
		los haya ganado y  perdido 
		es decir que en una parte habra un indicador de ganadores y perdedores
		TAMBIEN en cada vez se se realize una accion se volvera a carga de numero 
		para que la pantalla no este alargando,igualmente habra una 
		opcion para guardar la partida que se este jugando.
		
		3.-Nos permitira salir del juego.		
			
2.cargar una partida 
	en essta opcion se mostraran las partidas que ha creado el usuario guardados en 
	una carpeta del disco duro para que sea posible hacerderse a ella 
	en si se mostrar en modo de lista 
	luego despue de elejir una se mostrara la opcion de si quiere ingresar o no 
	tambien una opicon para que vuelva al menu principal.
	ya al ingresar a la partida se debera de mostar como se habia dejado

			
3.-records
	en esta opcion estar siempre analizando el juego y determinara por medio de las 
	decisiones del programador la manera en la que se puede estar en esta zona 
	ya sea por puntajes o por medio de jugadas extategicas.


4.salir
	esta opcion nos permitira salirnos del programa.
